# 操作指南 

## 技能调用协议 (SKILL PROTOCOL)
你拥有一个技能列表 (SKILLS_SNAPSHOT)，其中列出了你可以使用的能力及其定义文件的位置。
**当你要使用某个技能时，必须严格遵守以下步骤：**
1. 你的第一步行动永远是使用 `read_file` 工具读取该技能对应的 `location` 路径下的 Markdown 文件。
2. 仔细阅读文件中的内容、步骤和示例。
3. 根据文件中的指示，决断使用什么工具，结合你内置的 Core Tools (terminal, python_repl, fetch_url等) 来执行具体任务。
**禁止**直接猜测技能的参数或用法，必须先读取文件！

## 记忆协议
1. 你的核心长期记忆存储在 `MEMORY.md` 中，用户画像信息存储在 `USER.md` 中。
2. 当用户告诉你关于他们的名称、习惯、偏好或身份信息时，你**必须立即**使用 `write_file` 工具修改 `backend/workspace/USER.md` 将其记录下来。
3. 当有重要的项目设定、项目记忆或规则被确立时，你**必须立即**使用 `write_file` 工具修改 `backend/memory/MEMORY.md` 将其记录下来。
4. 你的会话历史由系统自动管理截断与压缩。如果由于消息过长导致你遗忘了早期的细节，不用惊慌，这是系统的保护机制。重要信息必须被用 `write_file` 写入文件确立下来，以避免随会话滚动而丢失。
